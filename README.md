# ⚡ Rock Paper Scissors – Pro Edition

A modern, animated, and interactive **Rock Paper Scissors web game** built using **HTML, CSS, and JavaScript**.
This project goes beyond basics with **advanced UI, animations, autoplay mode, keyboard controls, and persistent scoring**.

---

## 🚀 Features

* 🎮 Interactive gameplay (Rock / Paper / Scissors)
* ⚡ Advanced UI with **glassmorphism + neon effects**
* 🔄 Auto Play mode (AI vs AI simulation)
* ⌨️ Keyboard support

  * `R` → Rock
  * `P` → Paper
  * `S` → Scissors
* 💾 Score persistence using **localStorage**
* 🧠 Clean game logic with optimized structure
* 🏆 First to 3 wins system
* ✨ Animations:

  * Floating buttons
  * Glow effects
  * Pulse & ripple interactions

## ▶️ How to Run

### ✅ Option 1 (Recommended)

Open directly in browser:

```
index.html
```

### ✅ Option 2 (Better Experience)

Use Live Server in VS Code:

1. Install Live Server extension
2. Right-click `index.html`
3. Click **"Open with Live Server"**

## 🎯 Game Rules

* Rock beats Scissors
* Paper beats Rock
* Scissors beats Paper
* First to **3 wins** wins the match

---

## 🔄 Auto Play Mode

Click **Auto Play** to let the game play automatically.

Click again to **Stop**.
